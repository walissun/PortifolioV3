HABILIDADES

HTML

HTML (HyperText Markup Language) é a linguagem básica usada para criar e estruturar páginas na web. Ele define o conteúdo, como textos, imagens e links, que aparecem em um site.

Java Script

JavaScript é uma linguagem de programação que adiciona interatividade às páginas web. Com ele, é possível criar elementos dinâmicos, como sliders, formulários interativos e animações, melhorando a experiência do usuário.

CSS

CSS (Cascading Style Sheets) é a linguagem usada para estilizar páginas web. Com CSS, você pode alterar cores, fontes, layout e outros aspectos visuais de um site, tornando-o mais atraente e fácil de usar.
